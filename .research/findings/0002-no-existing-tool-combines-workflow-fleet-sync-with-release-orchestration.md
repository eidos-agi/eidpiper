## Claim

Existing tools cover parts but not the whole: repo-file-sync-action does file sync without drift analytics, GitHub reusable workflows reduce duplication but don't provide fleet auditing, release-please/semantic-release automate per-repo but not fleet-wide. No single tool does fleet-wide workflow management + release pipeline orchestration.
